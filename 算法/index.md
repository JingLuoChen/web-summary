# 算法
## 复杂度分析---算法性能的衡量指标
需要在不受外在因素影响的、大致的估计算法执行效率的方法---即使用复杂度分析的原因

## 如何表示复杂度
如何表示算法复杂度，具体来讲就是代码执行的时间、执行消耗的存储空间

## 常见时间复杂度
时间复杂度表示算法的执行时间与数据规模之间的增长关系
```
常量阶：O(1)
对数阶：O(logn)
线性阶：O(n)
线性对数阶：O(nlogn)
平方阶、立方阶、….、k次方阶：O(n2)、O(n3)、…、O(nk)
指数阶：O(2k)
阶乘阶：O(n!)
```

## 常见空间复杂度
空间复杂度表示算法的存储空间与数据规模之间的增长关系
